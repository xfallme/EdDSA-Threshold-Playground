Message Verification

Here you can verify a message and its signature using the group public key.

Please note that this verification is done using normal EdDSA and not the FROST protocol, since that is the whole point of running FROST with Ed25519/Ed448.

This means, any verifier that adheres to the EdDSA standard can verify the signature, without needing to know about FROST or the threshold scheme. For example, you can use my EdDSA playground.
